# K–12 Deep Learning Library

This is the curriculum map behind `src/content/k12-library.ts`. It contains 19 course pathways, 220+ sequenced units, and links to vetted free/open collections. It is a discovery catalog—not a redistribution of third-party books.

## How to use it

1. Choose a grade-band pathway.
2. Work through units in order or let a guardian assign a unit.
3. Open a listed provider resource.
4. Before downloading or bundling material, add the title-level attribution to the citation log.

## Licensing labels

- **Open-review-required**: the provider offers openly licensed material, but each selected book, version, and embedded asset must be checked before it is bundled.
- **Free-link-only**: it is available free at the provider; Learning World OS links out and does not copy it until permissions are recorded.

## Priority download packs

1. **K–5 Core Pack** — Core Knowledge Language Arts, Science, History & Geography, and Math units.
2. **Middle School STEM Pack** — CK-12 science/math texts plus carefully attributed PhET activities.
3. **High School Foundations Pack** — OpenStax Algebra, Statistics, Biology, Chemistry, Physics, Psychology, Economics, and U.S. Government.
4. **Primary Sources Pack** — Library of Congress teacher guides and public-domain source sets, with item-level rights checks.
5. **Computer Science Pack** — Code.org unit links and locally authored practice exercises; do not copy proprietary videos/artwork.
