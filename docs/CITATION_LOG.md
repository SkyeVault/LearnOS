# Learning World OS — Open Textbook Citation & License Log

Last reviewed: 2026-07-16

This log tracks source collections used to discover or package open educational resources. It is not a claim that every book in a collection has the same license. Every bundled title requires its own entry before release.

| ID | Source / publisher | Citation and canonical URL | License / reuse finding | How Learning World OS may use it |
| --- | --- | --- | --- | --- |
| OTL-001 | Open Education Network, Open Textbook Library | *Open Textbook Library*, https://open.umn.edu/opentextbooks/ | The Library describes titles as freely usable and adaptable under author/publisher licenses. It reports 1,760+ titles. Individual licenses vary. | Discovery only until a title-level license is captured. |
| OTL-002 | Open Education Network | *Open Textbook Library FAQ*, https://open.umn.edu/opentextbooks/faq | The Library defines an open textbook as openly licensed for free use and change; titles can be downloaded and adopted without extra paperwork. | Use its catalog as a title discovery source; preserve each book’s own attribution. |
| OTL-003 | Open Education Network | *Open Textbook Library Discovery*, https://open.umn.edu/opentextbooks/discovery | The collection exposes public catalog records in JSON/CSV/MARC and catalog data is CC0. | A future import job may use catalog metadata, but not treat CC0 metadata as a license for book content. |
| OST-001 | OpenStax / Rice University | *Licensing information for OpenStax textbooks*, https://help.openstax.org/s/article/Licensing-information-of-OpenStax-textbooks | Current OpenStax textbooks are CC BY-NC-SA 4.0; attribution must name OpenStax, title, and link to the free version. | May bundle/adapt only for noncommercial distribution with complete attribution and the same license on adaptations. |
| OST-002 | OpenStax / Rice University | *OpenStax Subjects*, https://openstax.org/subjects | OpenStax lists peer-reviewed, openly licensed, free texts across business, CS, science, social science, math, and more. | Discovery source; verify individual title/version before packaging. |
| OST-003 | OpenStax | *Biology 2e*, https://openstax.org/details/books/biology-2e | The title page identifies Biology 2e as CC BY-NC-SA 4.0. | Candidate for a reviewed, attributed science package. |
| LTX-001 | LibreTexts | *Copyright and Licensing — Algebra 1*, https://math.libretexts.org/Courses/Clackamas_Community_College/MTH_60%3A_Algebra_1/00%3A_Front_Matter/04%3A_Licensing | This specific book is CC BY-NC 4.0. LibreTexts book/page licenses vary. | Use only title-specific permissions; do not infer a universal LibreTexts license. |
| BCC-001 | BCcampus | *OER by Discipline Directory — General OER Repositories*, https://pressbooks.bccampus.ca/vccoerdiscipline/chapter/general-oer-repositories/ | Documents large open-textbook discovery collections; licenses vary by item. | Discovery only until title-level review. |
| WKB-001 | Wikibooks | *Using Wikibooks / Donating a Book to Wikibooks*, https://en.wikibooks.org/wiki/Using_Wikibooks/Donating_a_Book_to_Wikibooks | Wikibooks content is generally dual-licensed CC BY-SA 4.0 and GFDL unless noted. | May use with full contributor attribution and ShareAlike/GFDL compliance; review each page/history before reuse. |
